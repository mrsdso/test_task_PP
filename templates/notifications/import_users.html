Формат JSON файла

JSON файл должен содержать массив объектов со следующими полями:

[
  {
    "id": 1,
    "email": "user@example.com",
    "phone": "+1234567890",
    "telegram": "@username",
    "group": "admins"
  },
  ...
]
Описание полей:
  • id - уникальный числовой ID
  • email - email адрес пользователя
  • phone - номер телефона
  • telegram - Telegram username
  • group - название группы
Группы будут созданы автоматически, если их не существует.
Существующие пользователи будут обновлены по ID.
